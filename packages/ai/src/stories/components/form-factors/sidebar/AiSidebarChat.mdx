<Title>AI Sidebar Chat</Title>

The AI Sidebar Chat component is a form factor component that positions a slotted chatbot in a sidebar or fullscreen modal. It manages positioning and expand/collapse state while delegating all chat functionality to the slotted `ai-chatbot` component. When expanded, the chat is displayed in a fullscreen modal. When collapsed, it's displayed in a sidebar.

## Features

- **Composition-based**: Accepts slotted chatbot component for maximum flexibility
- **Built-in event handling**: Manages open/close events and expand/collapse interactions
- **Dual display modes**: Sidebar for normal view, fullscreen modal when expanded
- **Direct chatbot access**: Exposes slotted chatbot via `chatbot` property for programmatic control
- **Event bubbling**: All chatbot events bubble through unchanged

## When to Use

Use the AI Sidebar Chat component when you want:

- A sidebar container for your chatbot that can expand to fullscreen
- Standard sidebar positioning without custom logic
- Built-in modal transition for expanded view
- Direct control over the chatbot configuration

## Usage

```html
<forge-ai-sidebar-chat>
  <forge-ai-chatbot .adapter="\${adapter}" file-upload="enabled"> </forge-ai-chatbot>
</forge-ai-sidebar-chat>
```

### Programmatic Control

```javascript
const sidebarChat = document.querySelector('forge-ai-sidebar-chat');

// Control form factor
sidebarChat.show();
sidebarChat.close();
sidebarChat.expand(); // Switch to fullscreen modal
sidebarChat.collapse(); // Return to sidebar

// Access slotted chatbot
const chatbot = sidebarChat.chatbot;
chatbot.sendMessage('Hello!');
```
