The AI Chatbot Launcher provides a page-embedded chat experience featuring a centered welcome/hero view that transitions into a full conversation interface when the user sends their first message. It combines a visually engaging landing state with full chatbot functionality.

## Features

- **Welcome view**: Centered hero layout with customizable icon, title, description, and suggestions
- **Conversation view**: Full chat interface with header controls (clear, export, agent selection)
- **Animated transitions**: Smooth state changes between welcome and conversation modes

## When to Use

Use the AI Chatbot Launcher when you want:

- A dedicated landing page or section for AI chat
- A visually appealing entry point before conversation begins
- Multi-agent support with header-based agent switching
- Built-in welcome-to-conversation state management

For inline embedded experiences without a welcome view, consider `<forge-ai-chatbot>`. For floating/overlay patterns, use `<forge-ai-floating-chat>`.

## Usage Patterns

### Welcome Mode (Default)

Initial state displays a centered hero with icon, title, description, and optional suggestions. The prompt input is prominent and wrapped in a gradient container.

### Conversation Mode

After sending the first message, transitions to a full chat interface with:

- Header showing title, agent selector, and action buttons
- Scrollable message thread
- Bottom-anchored prompt area

## Customization

### Header Actions Slot

The `header-actions` slot allows you to add custom action buttons to the conversation header, positioned before the built-in header actions. These buttons appear in the conversation view after transitioning from the welcome screen.

```html
<forge-ai-chatbot-launcher .adapter="${adapter}">
  <forge-icon-button slot="header-actions" aria-label="History">
    <forge-icon name="history"></forge-icon>
  </forge-icon-button>
  <forge-icon-button slot="header-actions" aria-label="Settings">
    <forge-icon name="settings"></forge-icon>
  </forge-icon-button>
</forge-ai-chatbot-launcher>
```
