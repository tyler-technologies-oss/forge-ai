<Title>AI Floating Chat</Title>

The AI Floating Chat component is a form factor component that positions a slotted chatbot in a floating dialog. It manages positioning and expand/minimize state while delegating all chat functionality to the slotted `ai-chatbot` component.

## Features

- **Composition-based**: Accepts slotted chatbot component for maximum flexibility
- **Built-in event handling**: Manages expand/minimize/open/close events internally
- **Responsive behavior**: Adapts to different viewport sizes automatically
- **Direct chatbot access**: Exposes slotted chatbot via `chatbot` property for programmatic control
- **Event bubbling**: All chatbot events bubble through unchanged

## When to Use

Use the AI Floating Chat component when you want:

- A floating dialog container for your chatbot
- Standard floating chat positioning without custom logic
- Built-in responsive behavior and expand/collapse functionality
- Direct control over the chatbot configuration

## Usage

```html
<forge-ai-floating-chat>
  <forge-ai-chatbot .adapter="\${adapter}" enable-file-upload> </forge-ai-chatbot>
</forge-ai-floating-chat>
```

### Programmatic Control

```javascript
const floatingChat = document.querySelector('forge-ai-floating-chat');

// Control form factor
floatingChat.show();
floatingChat.close();
floatingChat.expand();
floatingChat.collapse();

// Access slotted chatbot
const chatbot = floatingChat.chatbot;
chatbot.sendMessage('Hello!');
```
