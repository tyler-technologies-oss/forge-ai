Displays terms of service, privacy policies, or other disclosures requiring user acknowledgment. Features optional icon, scrollable content, and pinned footer with action buttons.

<Canvas of={AiDisclaimerStories.Demo} />

## Usage

Use within form factor components to gate access to the AI assistant. Place content in the default slot, optionally provide an icon, and handle the agree/disagree events.

```html
<forge-ai-floating-chat open>
  <forge-ai-disclaimer @forge-ai-disclaimer-agree="${handleAgree}" @forge-ai-disclaimer-disagree="${handleDisagree}">
    <svg slot="icon" viewBox="0 0 24 24"><!-- icon path --></svg>
    <h2>Terms of Service</h2>
    <p>By using this AI assistant, you agree to our terms...</p>
  </forge-ai-disclaimer>
</forge-ai-floating-chat>
```

### Custom Button Text

```html
<forge-ai-disclaimer>
  <span slot="disagree-button-text">Decline</span>
  <span slot="agree-button-text">Accept</span>
  <p>Content here...</p>
</forge-ai-disclaimer>
```
