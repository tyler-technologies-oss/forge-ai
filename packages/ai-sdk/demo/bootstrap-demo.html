Floating Chat (CDN Import)

This page demonstrates the bootstrap script with CDN import.

The chatbot appears in the bottom-right corner of the screen.

Usage:

<script type="module">
  import { initChatbot } from '../dist/bootstrap.js';

  const api = await initChatbot({
    agentId: 'agent-9b3ff935-f32d-477b-ac45-ce2a3570b90c',
    baseUrl: 'http://localhost:3001',
    chatViewType: 'floating'
  });

  console.log('Chatbot ready!', api);
</script>

Events

Listen for the ready event:

window.addEventListener('tyler-ai-chatbot-ready', (event) => {
  console.log('API:', event.detail.api);
});
